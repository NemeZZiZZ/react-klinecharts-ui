## Why react-klinecharts-ui?

<CardGrid stagger>
  <Card title="Headless by design" icon="puzzle">
    No UI components ship with the library. You get a state provider and hooks —
    render them with Tailwind, shadcn, Radix, MUI or anything else.
  </Card>
  <Card title="Composable hooks" icon="seti:react">
    25+ focused hooks: periods, symbol search, indicators, drawing tools, order
    lines, alerts, replay, layouts, undo/redo and more.
  </Card>
  <Card title="Pro-grade overlays" icon="pencil">
    Drawing-tool overlay templates — Fibonacci, Elliott waves, Gann, harmonic
    patterns, position tools — ready to register on the chart.
  </Card>
  <Card title="TradingView-style indicators" icon="bars">
    Custom indicator templates (Ichimoku, SuperTrend, VWAP, HMA, MA Ribbon, TV
    MACD/RSI/BOLL) plus a TA math library.
  </Card>
</CardGrid>

## Quick taste

```tsx
import { KlinechartsUIProvider } from "react-klinecharts-ui";
import { KLineChart } from "react-klinecharts";

<KlinechartsUIProvider datafeed={datafeed} defaultSymbol={symbol} defaultTheme="dark">
  <MyTerminalUI />
</KlinechartsUIProvider>;
```
