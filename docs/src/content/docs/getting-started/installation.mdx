react-klinecharts-ui ships as ESM + CJS with TypeScript types. Install it
alongside its peer dependencies `react`, `react-dom` and `klinecharts`.

<Tabs syncKey="pkg">
  <TabItem label="npm">
    ```bash
    npm install react-klinecharts-ui klinecharts
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add react-klinecharts-ui klinecharts
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add react-klinecharts-ui klinecharts
    ```
  </TabItem>
</Tabs>

## Peer dependencies

| Package             | Required | Version              |
| ------------------- | -------- | -------------------- |
| `react`             | yes      | `>= 17.0.0`          |
| `react-dom`         | yes      | `>= 17.0.0`          |
| `klinecharts`       | yes      | `>= 10.0.0-beta3`    |
| `react-klinecharts` | optional | `>= 0.3.0`           |

`react-klinecharts` is only needed if you render the chart with its `<KLineChart>`
component or the `ChartCanvas` wrapper from `react-klinecharts-ui/chart`. If you
initialise the chart yourself with `klinecharts.init()`, you can skip it.

## Entry points

The package exposes three entry points:

```ts
// Main: provider, hooks, overlays, indicators, utils, data/constants
import { KlinechartsUIProvider, useIndicators } from "react-klinecharts-ui";

// Extensions: low-level registration + order line / depth overlays
import { registerExtensions, orderLine } from "react-klinecharts-ui/extensions";

// Chart: optional thin renderer wrapper (requires `react-klinecharts`)
import { ChartCanvas } from "react-klinecharts-ui/chart";
```
